jquery-matrixmultiplikation.js

matrix multiplication is a operation that produces a matrix from two matrices A and B. In more datail, if A is an n x m matrix and B is an m x p matrix, their product AB is an n x p matrix in which the m entries across a row of A are multiplied with the m entries down a columns of B and dummed to produce an entry of AB.
